# University ERP System

## Overview
The University ERP System is a centralized academic management platform designed to digitize and automate university administrative and educational processes. The system replaces manual workflows with a modern ERP-based solution that improves efficiency, data management, and communication between students, teachers, and administrators.

The project was developed using ERPNext for backend management and React.js for building responsive user interfaces.

---

# Features

## Authentication & Role Management
- Secure login system
- Role-based access control
- Separate dashboards for:
  - Administrators
  - Teachers
  - Students

---

## Academic Management
- Student registration management
- Course and module management
- Attendance tracking
- Grade management
- Academic report generation

---

## Attendance & Grades Automation
- Automated attendance calculations
- Grade processing and management
- Performance monitoring

---

## Dashboard & Analytics
- Interactive dashboards
- Attendance statistics

## API Integration
